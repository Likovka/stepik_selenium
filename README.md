# stepik_selenium

Выполненные задания для курса "Автоматизация тестирования с помощью Selenium и Python" на stepik

Ссылка на курс: https://stepik.org/course/575/
***
**Задания выполнялись для браузера Firefox.**

**Решением каждой задачи является вывод проверочного кода из всплывающего окна на консоль.**
***
## Модуль 1
Включает в себя пять практических задач на тему поиска элементов на странице.
1. Простой поиск полей для заполнения в форме регистрации.
2. Усложнённый вариант первой задачи: требуется найти ссылку на форму регистрации среди множества ссылок.
3. Заполнение множества обязательных полей за малое время.
4. Реализация поиска настоящей кнопки отправки на странице по XPath.
5. Заполнение только полей, отмеченных *.
## Модуль 2
Включает в себя восемь практических задач на тему изучения методов Selenium.
1. Подсчёт математической функции со спрятанным значением в тексте элемента страницы, использование checkbox и radiobutton.
2. Усложнённый вариант первой задачи: значение спрятано в атрибуте иконки.
3. Расчёт суммы чисел на странице, выбор значения из выпадающего списка.
4. Подсчёт математической функции со спрятанным значением в тексте элемента страницы, необходимо прокрутить страницу вниз.
5. Загрузка текстового файла file.txt, расположенного рядом с кодом задачи.
6. Подсчёт математической функции со спрятанным значением в тексте элемента страницы, обязательно подтверждение во всплывающем окне.
7. Подсчёт математической функции со спрятанным значением в тексте элемента страницы, переключение на другую вкладку.
8. Ожидание появления нужного текста на странице.
